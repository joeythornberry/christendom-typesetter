Christendom Typesetter

a streamlined typesetter that turns plain text into fully-formatted academic papers

Step One: Open your paper's directory

Each paper that uses Christendom Typesetter must have its own directory (file folder). Create one or more .txt files in that directory, and you can use Christendom Typesetter to combine them into a paper. (The button above can also create a directory if you don't already have one.)

Step Two: Write!

Christendom Typesetter will format your paper for you. Click the button above to open the live preview, and use a text editor1 of your choice to open one of the files in the directory you picked and start writing. When you're done, Christendom Typesetter will compile your paper into a PDF.

1You don't use this program to actually write your paper - it just makes it look nice. You can use any text editor you want.

Microsoft Word and Google Docs are not text editors - they are word processors, and this program is designed to replace them. Use Notepad (Windows) or TextEdit (Mac) instead. I use Neovim.
